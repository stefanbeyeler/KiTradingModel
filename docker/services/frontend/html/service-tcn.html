TCN-Pattern Service

Temporal Convolutional Network - Chart Pattern Erkennung mit Deep Learning

🧠 Beschreibung

Der TCN-Pattern Service nutzt Temporal Convolutional Networks (TCN) zur automatischen Erkennung von Chart-Patterns in Finanzzeitreihen. TCNs sind speziell für sequentielle Daten optimiert und bieten gegenüber RNNs den Vorteil paralleler Verarbeitung und stabilerem Training.

Der Service erkennt 16 verschiedene Chart-Pattern-Typen, die von Tradern weltweit zur technischen Analyse verwendet werden. Die Erkennung erfolgt in Echtzeit und kann für verschiedene Zeitrahmen konfiguriert werden.

⚙️ Funktionsweise

Der TCN-Algorithmus funktioniert durch:

  • Dilated Causal Convolutions: Erweiterte Konvolutionen erfassen lange Abhangigkeiten
  • Residual Connections: Stabile Gradienten für tiefe Netzwerke
  • Multi-Scale Features: Pattern werden auf verschiedenen Zeitskalen erkannt
  • Attention Mechanismus: Fokus auf relevante Zeitpunkte im Chart
GPU-Beschleunigung: Der Service nutzt NVIDIA CUDA für Echtzeit-Inferenz. Die Pattern-Erkennung dauert typischerweise unter 100ms pro Symbol.

📦 Modell-Architektur

Die TCN-Architektur des Pattern-Erkennungsmodells:

  • Input Layer: OHLCV-Daten normalisiert auf [0,1]
  • TCN Blocks: 4 Schichten mit Dilation [1, 2, 4, 8]
  • Hidden Channels: 64 pro Schicht
  • Kernel Size: 7 für breite rezeptive Felder
  • Output Layer: Multi-Label Klassifikation (16 Pattern + Confidence)
Mindestdaten: Für eine zuverlässige Pattern-Erkennung werden mindestens 100 Kerzen historischer Daten benötigt. Langere Zeitreihen verbessern die Genauigkeit.

📈 CRT - Candle Range Theory

Die CRT-Implementierung erkennt Liquidity Purges an Key Sessions und generiert Trading-Signale basierend auf der Candle Range Theory. Key Sessions sind: 1 AM, 5 AM und 9 AM EST.

CRT State Machine: RANGE_DEFINED → PURGE_ABOVE/BELOW → SIGNAL_LONG/SHORT
