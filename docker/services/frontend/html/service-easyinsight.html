EasyInsight API

Externe Datenquelle für OHLCV-Marktdaten und Symbol-Informationen

📊 Beschreibung

Die EasyInsight API ist die zentrale externe Datenquelle des KI Trading Model Systems. Sie liefert historische und aktuelle Marktdaten (OHLCV - Open, High, Low, Close, Volume) für verschiedene Finanzinstrumente wie Kryptowährungen, Forex-Paare, Indizes und Edelmetalle.

Die Daten werden in einer TimescaleDB gespeichert, einer zeitreihenoptimierten PostgreSQL-Erweiterung, die besonders für Finanzdaten mit hoher Schreiblast und komplexen Zeitabfragen geeignet ist.

⚙️ Funktionsweise

Der Service funktioniert wie folgt:

  • Datensammlung: Automatisches Abrufen von Marktdaten von verschiedenen Börsen (Kraken, etc.)
  • Speicherung: Persistente Speicherung in TimescaleDB mit optimierter Zeitreihen-Kompression
  • API-Bereitstellung: RESTful API mit Swagger-Dokumentation für einfachen Zugriff
  • Kategorisierung: Automatische Kategorisierung der Symbole (Forex, Crypto, Indices, Metals)
Hinweis: Dieser Service läuft auf einem separaten Server (10.1.19.102) und wird vom KI Trading Model über das Netzwerk angesprochen. Die Latenz wird im Dashboard angezeigt.

🔗 Wichtige API-Endpunkte

GET /api/symbols Liste aller verfügbaren Symbole
GET /api/ohlcv/{symbol} OHLCV-Daten für ein Symbol
GET /api/symbols/{symbol}/info Detailinformationen zu einem Symbol
GET /api/health Service Health-Check

📦 Datenstruktur

Die OHLCV-Daten werden in folgendem Format bereitgestellt:

  • timestamp: Zeitstempel des Datenpunkts (UTC)
  • open: Eröffnungspreis der Periode
  • high: Höchstkurs der Periode
  • low: Tiefstkurs der Periode
  • close: Schlusskurs der Periode
  • volume: Handelsvolumen der Periode
Wichtig: Die Datenqualität und -verfügbarkeit hängt von der jeweiligen Börse ab. Nicht alle Symbole haben Daten für alle Zeiträume verfügbar.
