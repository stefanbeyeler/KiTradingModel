EasyInsight API

TwelveData-kompatible API für OHLCV-Marktdaten und technische Indikatoren

📊 Beschreibung

Die EasyInsight API ist die zentrale externe Datenquelle des KI Trading Model Systems. Mit der neuen TwelveData-kompatiblen API (v2.0) liefert sie historische und aktuelle Marktdaten sowie technische Indikatoren für alle Timeframes (M1 bis MN).

Die neue API-Struktur ermöglicht eine nahtlose Integration als Fallback für TwelveData und bietet identische Endpunkt-Strukturen für OHLCV-Daten und Indikatoren.

NEU in v2.0: TwelveData-kompatible Endpoints mit Unterstützung für alle Timeframes (M1, M5, M15, M30, H1, H4, D1, W1, MN) und individuelle Indikator-Endpoints.

⚙️ API-Architektur

Die neue API folgt dem TwelveData-Format:

  • /time_series/{symbol}: OHLCV-Daten für alle Timeframes mit konfigurierbarem Intervall
  • /rsi/{symbol}, /macd/{symbol}, etc.: Individuelle Indikator-Endpoints
  • /indicators/{symbol}: Mehrere Indikatoren in einem Request
  • /quote/{symbol}: Echtzeit-Quote mit aktuellem Preis
  • /symbols: Symbol-Liste mit Typ-Filterung
Hinweis: Der Data Service nutzt EasyInsight automatisch als Fallback, wenn TwelveData nicht verfügbar ist oder ein Symbol nicht unterstützt wird.

🔗 OHLCV-Endpunkte

Time Series (Kerzen-Daten)

GET /time_series/{symbol} OHLCV-Daten (alle Timeframes)
GET /quote/{symbol} Echtzeit-Quote
GET /price/{symbol} Aktueller Preis (lightweight)

Symbol-Verwaltung

GET /symbols Liste aller Symbole
GET /status API-Status und Health

📈 Technische Indikatoren

Folgende Indikatoren sind als individuelle Endpoints verfügbar:

RSI - Relative Strength Index
MACD - Moving Average Convergence
BBands - Bollinger Bands
Stoch - Stochastic Oscillator
ADX - Average Directional Index
ATR - Average True Range
EMA - Exponential Moving Avg
SMA - Simple Moving Average
CCI - Commodity Channel Index
Ichimoku - Ichimoku Cloud
Strength - Multi-TF Strength (proprietär)

Indikator-Endpoints

GET /rsi/{symbol} RSI (time_period=14)
GET /macd/{symbol} MACD (12, 26, 9)
GET /bbands/{symbol} Bollinger Bands (20, sd=2)
GET /stoch/{symbol} Stochastic (14, 3, 3)
GET /adx/{symbol} ADX mit +DI/-DI
GET /indicators/{symbol} Multiple Indikatoren (Batch)
GET /strength/{symbol} Multi-Timeframe Strength

🕑 Unterstützte Timeframes

Mit API v2.0 werden alle Standard-Timeframes unterstützt:

  • 1min, 5min, 15min, 30min: Intraday-Daten
  • 1h, 4h: Stündliche Daten
  • 1day, 1week, 1month: Tägliche/Wöchentliche/Monatliche Daten
Interval-Parameter: Verwende TwelveData-Format (1min, 5min, 15min, 30min, 1h, 4h, 1day, 1week, 1month) bei allen API-Aufrufen.

📦 Response-Format

OHLCV-Responses folgen dem TwelveData-Format:

  • meta: Metadaten (symbol, interval, type)
  • values: Array mit OHLCV-Datenpunkten
  • status: "ok" oder "error"

Jeder Datenpunkt enthält:

  • datetime: Zeitstempel (YYYY-MM-DD HH:MM:SS)
  • open, high, low, close: Preisdaten als Strings
  • volume: Handelsvolumen
